# Mean-Variance-Standard Deviation Calculator

FreeCodeCamp Data Analysis with Python - Project 1

## Project Description
This project calculates the mean, variance, standard deviation, max, min, and sum of a 3x3 matrix using NumPy.

## Features
- Converts a list of 9 numbers into a 3x3 matrix
- Calculates statistics along rows, columns, and flattened matrix
- Returns results in a structured dictionary format
- Handles errors for invalid input

## Technologies Used
- Python
- NumPy
